# Modern_Pandas
Just playing around with Pandas

## Contents

- `modern_pandas_exploration.ipynb` - A comprehensive Jupyter notebook exploring modern Pandas features including data manipulation, analysis, and visualization

## Getting Started

### Installation

Install the required dependencies:

```bash
pip install -r requirements.txt
```

### Running the Notebook

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Then open `modern_pandas_exploration.ipynb` in your browser.

## Features

The notebook covers:
- Creating and inspecting DataFrames
- Data selection and filtering techniques
- Data manipulation operations
- Grouping and aggregation
- Time series analysis
- Data visualization with matplotlib
